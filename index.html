Cloud Computing and Virtualization Program @ Atlantic Technical College



The Cloud Computing & Virtualization program prepares students for entry-level employment cloud support fields, this program can be used to provide supplemental training for people seeking to update their skills. Students gain technical expertise in core cloud design and infrastructure, using cloud technologies, and prepares them to monitor, and maintain virtual computer networks.

The curriculum is delivered through instructor-led classes, knowledge assessments, and real-world hands-on labs. Students will also have access to course manuals, online knowledge assessments, and a free practice certification exam.Our 5-course program covers all the necessary topics to prepare you for the CompTIA IT Fundamentals+, CompTIA Cloud+, and AWS Cloud Practitioner exams.

ATC's Cloud Computing & Virtualization program is a member of Amazon's AWS Academy , and through this program, students are prepared to take the AWS Certified Cloud Practitioner certification exam. AWS Academy courses help students develop in-demand cloud computing skills by providing hands-on experience in the AWS Cloud and lessons based on actual industry scenarios. AWS Academy courses and learning resources are aligned to industry-recognized AWS Certifications.


Information Technology Assistant (OTA0040)

Course Assignments & Due Dates

The first course of the program will start on the basics. Students will develop an understanding of the workings of microprocessors and digital computers, as well as operating systems and application software. Students will be able to demonstrate competence using computer networks, internet and online databases to facilitate collaborative or individual learning and communication to accomplish job objectives and enhance workplace performance.


Computer Support Assistant (EEV0504)


Students will learn to diagnose and troubleshot common computer problems and system malfunctions. Students will also learn the principles of motherboards, processors and memory in desktop and server computer systems and be able to identify and describe network concepts and terminology. Additionally, they will be able to determine whether a computer is networked or not and acquire knowledge of the ramifications of repairs when a computer is networked.


Network Support Technician (CTS0026)


This course will prepare students to perform installation and configuration activities. Students will be able to demonstrate proficiency in troubleshooting and diagnosing, Perform installation and configuration activities, and computer networks as well as troubleshooting hardware devices and drivers managing, monitoring, and optimizing system performance, reliability and availability of network connections


Cloud Analyst (CTS0054)


This course will provide students with a solid understanding of cloud computing concepts. In this course students will have the opportunity to see how those concepts are implemented on AWS, Azure and GCP. Students will be able to verify their understanding of virtualization concepts through labs in the various platforms.


Cloud Virtualization Specialist (CTS0056)


The course will build on the solid understanding of cloud computing concepts and dive into Amazon Web Services (AWS), security, architecture, pricing, and support. The curriculum utilized is developed and maintained by AWS and taught by AWS Academy accredited educators. Students will be able to identify and evaluate components of cloud based services.
